iconfont图标引用

发表于2022-12-04|更新于2024-08-10|Hexo

此处省略从iconfont 将图标添加到项目等基础步骤，请自行百度

iconfont 图标有三种引入方式, 因为自己图标库有彩色图标所以之前使用的是 symbol 引用方式, 但是多色图标占比很少,多用又反而觉得复杂了,所以采用了 Font Class 引用方式
但是!!!考虑到有部分想使用彩色图标的朋友小伴, 所以在某些步骤会同时写入😀

一、复制 iconfont 代码

在项目处选择 Font Class/ symbol 引用方式, 可点击链接下载, 复制文件内容保存在本地或者点此复制代码引用, 区别如下:

保存本地: 可在本地文件修改基本样式, 更新图标库需要重新下载替换代码, 会替换原有修改
复制链接: 引用更加方便, 更新图标库只需更新引用链接即可

二、引入

在 _config.butterfly.yml 中引入:

head: 引入Font Class CSS 文件
bottom: 引入symbol 的 js 文件

此步仅为 symbol 引入方式需要, Font Class 无需理会
在 [mybolg] > themes > butterfly > source > css > _cutom > cutom.css 中增加

三、使用图标

复制对应的 icon 代码，并在 _config.butterfly.yml 中对应位置输入即可
