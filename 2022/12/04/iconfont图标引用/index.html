iconfontеӣҫж Үеј•з”Ё

еҸ‘иЎЁдәҺ2022-12-04|жӣҙж–°дәҺ2024-09-20|Hexo

жӯӨеӨ„зңҒз•Ҙд»Һiconfont е°Ҷеӣҫж Үж·»еҠ еҲ°йЎ№зӣ®зӯүеҹәзЎҖжӯҘйӘӨпјҢиҜ·иҮӘиЎҢзҷҫеәҰ

iconfont еӣҫж Үжңүдёүз§Қеј•е…Ҙж–№ејҸ, еӣ дёәиҮӘе·ұеӣҫж Үеә“жңүеҪ©иүІеӣҫж ҮжүҖд»Ҙд№ӢеүҚдҪҝз”Ёзҡ„жҳҜ symbol еј•з”Ёж–№ејҸ, дҪҶжҳҜеӨҡиүІеӣҫж ҮеҚ жҜ”еҫҲе°‘,еӨҡз”ЁеҸҲеҸҚиҖҢи§үеҫ—еӨҚжқӮдәҶ,жүҖд»ҘйҮҮз”ЁдәҶ Font Class еј•з”Ёж–№ејҸ
дҪҶжҳҜ!!!иҖғиҷ‘еҲ°жңүйғЁеҲҶжғідҪҝз”ЁеҪ©иүІеӣҫж Үзҡ„жңӢеҸӢе°Ҹдјҙ, жүҖд»ҘеңЁжҹҗдәӣжӯҘйӘӨдјҡеҗҢж—¶еҶҷе…ҘрҹҳҖ

дёҖгҖҒеӨҚеҲ¶ iconfont д»Јз Ғ

еңЁйЎ№зӣ®еӨ„йҖүжӢ© Font Class/ symbol еј•з”Ёж–№ејҸ, еҸҜзӮ№еҮ»й“ҫжҺҘдёӢиҪҪ, еӨҚеҲ¶ж–Үд»¶еҶ…е®№дҝқеӯҳеңЁжң¬ең°жҲ–иҖ…зӮ№жӯӨеӨҚеҲ¶д»Јз Ғеј•з”Ё, еҢәеҲ«еҰӮдёӢ:

дҝқеӯҳжң¬ең°: еҸҜеңЁжң¬ең°ж–Үд»¶дҝ®ж”№еҹәжң¬ж ·ејҸ, жӣҙж–°еӣҫж Үеә“йңҖиҰҒйҮҚж–°дёӢиҪҪжӣҝжҚўд»Јз Ғ, дјҡжӣҝжҚўеҺҹжңүдҝ®ж”№
еӨҚеҲ¶й“ҫжҺҘ: еј•з”ЁжӣҙеҠ ж–№дҫҝ, жӣҙж–°еӣҫж Үеә“еҸӘйңҖжӣҙж–°еј•з”Ёй“ҫжҺҘеҚіеҸҜ

дәҢгҖҒеј•е…Ҙ

еңЁ _config.butterfly.yml дёӯеј•е…Ҙ:

head: еј•е…ҘFont Class CSS ж–Үд»¶
bottom: еј•е…Ҙsymbol зҡ„ js ж–Үд»¶

жӯӨжӯҘд»…дёә symbol еј•е…Ҙж–№ејҸйңҖиҰҒ, Font Class ж— йңҖзҗҶдјҡ
еңЁ [mybolg] > themes > butterfly > source > css > _cutom > cutom.css дёӯеўһеҠ
