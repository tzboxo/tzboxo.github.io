Note标签样式修改

发表于2022-12-04|更新于2022-12-04
|阅读量:|评论数:

博客的样式是基于Butterfly主题的原Note样式修改, 其中的\e62a为阿里图标项目引用, 阿里图标引用方法请查阅相关文章:

一、开启主题 Note 样式

在 _config.butterfly.yml 文件选择默认 Note 样式

二、修改样式

themes > butterfly > sources > css >var.styl

修改描边/投影样式 (随意), themes > butterfly > sources > css > _tags >note.styl

三、书写语法

default 默认

info 信息

primary 主要

success 成功

warning 警告

danger 危险
