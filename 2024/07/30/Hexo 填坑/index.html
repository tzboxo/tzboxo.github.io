二、npm i 命令提示错误

最后定位在了 CERT_HAS_EXPIRED 这个错误代码上。

原因：

遇到 CERT_HAS_EXPIRED 错误通常意味着尝试连接的服务器的 SSL 证书已过期。这个问题可能出现在你使用的 npm 镜像上（我用的是淘宝镜像）

解决方案：

方法 1：临时忽略 SSL 证书验证

作为临时解决方案，你可以通过设置 npm 配置来忽略 SSL 证书验证。请注意，这种方法会降低安全性，不推荐作为长期解决方案。
